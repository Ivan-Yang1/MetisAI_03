# MetisAI 项目需求文档

## 项目概述

MetisAI 是一个基于 OpenHands 架构的 AI 驱动开发平台，允许用户通过 AI 智能体完成软件开发任务。

## 核心功能

### 1. 智能体系统

**需求描述**：
- 提供多个预定义的 AI 智能体（如 CodeActAgent）
- 支持智能体的创建、启动、停止、删除操作
- 支持多种 LLM 模型配置（Claude, GPT, Mistral 等）
- 智能体可以执行各种开发任务

**优先级**：高

### 2. 会话管理

**需求描述**：
- 支持会话的创建、删除、查询
- 提供会话历史查看功能
- 支持会话状态管理（活动、已完成）
- 支持会话参数配置

**优先级**：高

### 3. 消息交互

**需求描述**：
- 支持用户与智能体之间的消息交流
- 支持文本消息和代码块
- 支持消息历史记录和搜索
- 实时显示智能体响应

**优先级**：高

### 4. 代码执行

**需求描述**：
- 支持在沙箱环境中执行命令
- 支持文件操作（创建、读取、修改、删除）
- 支持代码运行和调试
- 提供代码高亮显示

**优先级**：高

### 5. 沙箱环境

**需求描述**：
- 提供安全的执行环境（Docker 容器）
- 支持文件持久化和权限管理
- 提供资源限制和超时机制
- 支持浏览器环境（Playwright）

**优先级**：高

### 6. 前端界面

**需求描述**：
- 直观的用户界面
- 响应式设计，支持多种设备
- 实时反馈和状态显示
- 简洁的操作流程

**优先级**：高

## 可选功能

### 7. 用户管理（可选）

**需求描述**：
- 用户注册、登录、注销
- 用户资料管理
- 用户权限控制
- 密码管理

**优先级**：中

### 8. 高级功能（可选）

**需求描述**：
- 项目管理功能
- 任务调度和自动化
- 集成 CI/CD 流程
- 高级分析和报告

**优先级**：低

## 技术架构

### 前端技术栈

- React 19
- TypeScript
- Redux Toolkit
- React Router
- Socket.IO
- Axios

### 后端技术栈

- Python 3.12-3.13
- FastAPI
- SQLAlchemy
- Pydantic
- JWT（可选）
- python-socketio
- litellm
- Docker

## 数据存储

- PostgreSQL（或 SQLite 用于开发）
- 文件系统存储

## 部署方式

- Docker Compose 本地部署
- Kubernetes（生产环境）

## 开发阶段

**第一阶段（基本功能）**：
- 项目基础配置
- 智能体系统架构
- 会话管理
- 消息交互
- 代码执行
- 沙箱环境
- 前端界面

**第二阶段（可选功能）**：
- 用户管理
- 高级功能
- 优化与测试

## 成功标准

1. 用户可以成功创建并运行智能体
2. 智能体可以执行基本开发任务
3. 用户可以与智能体进行有效的交流
4. 系统提供可靠的沙箱执行环境
5. 界面友好且响应迅速
